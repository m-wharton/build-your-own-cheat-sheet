CSS selectors

Selectors Examples Description
Universal * The universal selector selects all elements of any type.
Type h1, p, table The type selector matches the type of the element in the HTML document.
Class .container To select this element using CSS, we can create a ruleset with a class selector of .container
ID #page-title To select this element using CSS, we can create a ruleset with a class selector of #page-title
Attribute [href] The attribute selector can be used to target HTML elements that already contain attributes. Elements of the same type can be targeted differently by their attribute or attribute value. This alleviates the need to add new code, like the class or id attributes.
Pseudo-class p:hover The appearance of certain elements can change, or be in a different state, after certain user interactions.
